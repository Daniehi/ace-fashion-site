# 👗 Ace Fashion Studio

A modern luxury fashion landing page built with:

- ⚡ Vite
- ⚛️ React
- 🟦 TypeScript
- 🎨 Custom CSS (no frameworks)

---

## ✨ About the Project

Ace Fashion Studio is a premium fashion landing page designed to represent a modern luxury brand.  
It focuses on minimal design, strong typography, and elegant visuals.

This project is built as a practice portfolio-level UI for a fashion brand concept.

---

## 🚀 Features

- Luxury hero section with full-screen background
- Smooth animations (fade-in effects)
- Responsive design (mobile + desktop)
- Modern navigation bar
- Fashion editorial layout
- Clean footer section
- Dark luxury theme with gold accents

---

## 🧱 Tech Stack

- React 18
- TypeScript
- Vite
- CSS3 (custom styling)
